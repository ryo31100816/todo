Todoアプリ

Ⅰ作成の趣旨
・PHPを使用した開発に必要な知識の習得

Ⅱできること
　・Todoリストの作成、編集、削除
　・TodoリストのCSV出力

Ⅱ使用言語・環境
・PHP7.4
・JQuery
・HTML/CSS
・Docker　LEMP環境
・AWS EC2（Ubuntu 18.04）

Ⅲテーブル
　・Users

　・Todos


Ⅳ実装機能、使用技術
・CRUD機能
・User作成・ログイン・ログアウト機能
・User作成時のメール認証
・CSV抽出機能
・JSによるDOM操作
・Ajaxによる非同期処理
・CSSでのレイアウト作成

Ⅴ力を入れた点
・トークンによるUser認証
　・CSV出力のバックグラウンド実行
